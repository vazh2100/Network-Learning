192.168.10.1/24
Маска подсети делит ip адрес на непосредственно сеть и на диапазон хостов.

| Маска /N | Кол-во адресов в подсети | Комментарий / usable адреса               |
|----------|--------------------------|-------------------------------------------|
| /32      | 1                        | Только один адрес, обычно для хоста       |
| /31      | 2                        | Point-to-point, оба адреса usable         |
| /30      | 4                        | 2 usable (для хоста, 1 сеть, 1 broadcast) |
| /29      | 8                        | 6 usable                                  |
| /28      | 16                       | 14 usable                                 |
| /27      | 32                       | 30 usable                                 |
| /26      | 64                       | 62 usable                                 |
| /25      | 128                      | 126 usable                                |
| /24      | 256                      | 254 usable, последний октет               |
| /23      | 512                      | 510 usable                                |
| /22      | 1024                     | 1022 usable                               |
| /21      | 2048                     | 2046 usable                               |
| /20      | 4096                     | 4094 usable                               |
| /19      | 8192                     | 8190 usable                               |
| /18      | 16384                    | 16382 usable                              |
| /17      | 32768                    | 32766 usable                              |
| /16      | 65536                    | 65534 usable, последние 2 октета          |

Если зона адреса под хосты в двоичном виде занята единицами, то это широковещательная отправка - все участники сети
получат этот пакет. Например, `192.168.10.255\24.`

Частные сети(локальные)
10.0.0.0/8 - класс A
172.16.0.0/12 - класс B
192.168.0.0/16 - класс C

`169.254.0.0` - подсеть локальных адресов связи. Используются для автоматического присвоения IP, если нет DHCP-сервера.
Применяются только внутри локальной сети, пакеты не маршрутизируются через роутеры, в интернет выйти не смогут. Только
L2, L3 не работает.
