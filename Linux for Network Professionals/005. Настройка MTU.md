`PDU`, `frame` - Блок Данных Протокола, кадр.
`MTU` - Максимальная Единица Передачи - это размер самого большого блока данных протокола, кадра.
`MSS` - Максимальный Размер Сегмента - предельная полезная нагрузка пакета.

В Ethernet MTU по умолчанию 1500 байт.

Больший размер обычно используют в местах с быстрым каналом 1-10 Гбит, iSCSI протокол(предоставление жесткого диск
через сеть на уровне блоков).

Приложение может установить для пакета бит DF - не фрагментировать. Если такой пакет больше размера кадра, случится
ошибка.

Уменьшение размера кадра встречается, когда пакеты инкапсулируются (виртуальные интерфейсы туннели для VPN), также при
спутниковой передаче.

`sudo nmcli connection modify Spusk_5.0 802-11-wireless.mtu 2000` - изменение размера MTU для соединения.
