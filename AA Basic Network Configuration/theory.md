`default via 10.42.41.131 dev wlan0 proto dhcp src 10.42.41.1 metric 600`
Если пакет направляется по адресу, которого нет в таблице маршрутизации, то он будет направлен в шлюз(gateway) по
умолчанию.
Таблица маршрутов всегда отдаёт приоритет «наиболее специфичному» маршруту — тому, который максимально точно совпадает с
ip-адресом назначения.
Хост предполагает, что default принадлежит роутеру, который знает куда дальше отправить пакет.

`10.42.41.0/24 dev wlan0 proto kernel scope link src 10.42.41.1 metric 600 `
Вторая строка - это маршрут Link Local Address

`default`  - это 0.0.0.0
